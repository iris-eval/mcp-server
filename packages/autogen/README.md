# @iris-eval/autogen

Auto-trace AutoGen multi-agent conversations with [Iris](https://iris-eval.com) — MCP-native eval and observability.

> **Status:** Scaffolded. Full integration coming soon. Star the repo to get notified.

## Planned Features

- Automatic tracing of agent-to-agent conversations
- Per-agent span tracking with message flow
- Tool call capture with input/output
- Token usage and cost per conversation turn
- Output quality evaluation per agent response
- Conversation topology visualization
